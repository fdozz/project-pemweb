### Prerequisites
- Node.js 18+ 
- npm atau yarn

### Installation

1. **Clone repository**
```bash
git clone https://github.com/fdozz/project-pemweb.git
cd project-pemweb
```

2. **Install dependencies**
```bash
npm install
```

3. **Setup database**
```bash
npm run seed
```

4. **Start development server**
```bash
npm start
```

5. **Buka browser**
```
http://localhost:3005
```
